Annex E (informative) Vendor identification

This annex contains the list of SCSI-2 vendor identifications (see table E.1) as of the date of this document. The purpose of this list is to help avoid redundant usage of vendor identifications. Task Group X3T9.2 of Accredited Standards Committee X3 maintains an informal list of vendor identifications currently in use. Please contact the chairman of X3T9.2 prior to using a new vendor identification to avoid conflicts.
