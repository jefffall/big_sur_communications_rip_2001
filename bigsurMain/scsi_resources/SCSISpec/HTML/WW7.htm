What's New?

Here is a list of recent changes to my HTML SCSI-2 Specification:

07-Mar-96

  • Added a link to Dal Allan's SCSI-1 vs. SCSI-2 document. Thanks to Lars-Owe Ivarsson for hosting that article and correcting me on its attribution.
  • Added a link to ANDATACO's white paper, The SCSI Challenge.

22-Feb-96

  • Added a version of the document using HTML frames (and made it the default version).
  • Removed color text and background from front page (but left texture).
  • HTML source code tidied up so that document is more usable with non-graphical browsers (e.g. lynx).
  • Finally tidied up colors/crops on figures.

14-Feb-96

  • Added a new access counter to more reliably record visitors (NOTE: the counter only records accesses through the front page(s), so to calculate the total number of hits on this server, multiply this number by your guess at the average number of pages accessed per visitor) .
  • Added color and texture to front page What do you think?

07-Feb-96

  • I finally got around to hyperlinking the SCSI-2 index This is now a very useful page for finding information (maybe more useful than my search page)
