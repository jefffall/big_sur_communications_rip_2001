SCSI-2 Specification


This is a transcription into HTML of the X3T9.2 Committee's draft ANSI specification of the SCSI-2 standard, revision 10L. SCSI-2 (the Small Computer System Interface - 2) was approved by the American National Standards Institute's Board of Standards Review on January 31, 1994. SCSI-2 will be designated X3.131-1994. The document was at revision 10L when it was approved.

This transcription was carried out by Gary Bartlett (last modified on Friday, 22-Mar-96 16:24:21 PST) using Meng Weng Wong's txt2html converter and lots of manual cutting and pasting from the original ASCII document.

DISCLAIMER: Please note that SCSI-2 Rev 10L files are NOT the ANSI SCSI-2 Standard. These files are the X3T9 Committee draft document prior to the approval and publication of SCSI-2 by ANSI. There were various editorial changes made during the publication process and there were several changes made to update reference standards in the density code and medium type tables.

ANSI permits X3 to make committee drafts available electronically, provided it is made clear that these files are not the ANSI Standard. If you want the final ANSI SCSI-2 standard in electronic format, please contact Global Engineering Documents (800-854-7179 or 303-792-2181); they plan to release a CD-ROM product including SCSI-2 in the Summer of 1995.
