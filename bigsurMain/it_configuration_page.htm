HP 4100N Printer Configuration

for Windows 2000 only

for windows 98, Windows ME or a laptop see below this article

We are connecting to the printer directly using TCP/IP.  Currently, there is no unix server to spool to the printer. And, that's fine.  The print jobs are spooled up on your Windows 2000 Professional PC and transferred via a TCP/IP port to the EIO network card on the HP 4100N printer. Maybe more than you wanted to know.

How to do it? Here are the steps..

  1. Open the Printers folder from the control panel

  2. Now top left do FILE and then select SERVER PROPERTIES

  3. Select the PORTS tab.

  4. Click ADD PORTS

  5. Highlight Standard TCP/IP Port by clicking the mouse ONCE on it.

  6. Now click once on NEW PORT - TCP/IP Wizard presents itself

  7. Click on NEXT

  8. For printer name or IP address use this IP address: 10.1.2.1

  9. For Port name use HP LJ 4100N

  10. Click on NEXT

  11. Now CLOSE or FINISH

  12. Next phase - Get the HP printer driver onto your computer by clicking [HERE]  DOWNLOAD HP 4100N driver (!! For Windows 2000 ONLY !!) For Windows ME, see next article below.

  13. Save the driver to somewhere on your hard drive where you will remember where you put it.

  14. Find where you put the file on your hard drive and rapidly click twice on it with the mouse.

  15. The driver packed exe file will expand and unzip its contents to lj828en on your C:\ drive.  That's OK, just remember where that is. So, all the driver files for the printer are now at C:\lj828en on your C:\ hard disk, OK?

  16. Now, go back to the control panel and find the Printers folder. Double click the folder to open it.

  17. Now hit ADD PRINTER

  18. Add Printer Wizard presents itself. Hit NEXT with mouse.

  19. Now, only allow "Local Printer" to be selected. Do not allow "Automatically detect ..." or "network printer" to be selected.. (whew - thanks)

  20. Now NEXT with the mouse.

  21. A list of ports is presented. Make sure "Use following port" is selected.

  22. From the list of ports, find the port you made in step 8 above.

  23. Highlight port called "HP LJ 4100N" which is near the end of the list.

  24. Now you see a bunch of names of printers presented.  Not to worry. You have the printer driver and that will automatically figure out the name of the printer - so you don't have to.. Select "HAVE DISK".

  25. Install from Disk menu comes up. Remember that disk path I told you to remember? Put C:\lj828en into the "Copy Manu files from" area. and hit NEXT

  26. Note the name of the printer driver it found from your directory..It is called HP Laserjet 4100 PCL 6. That's fine. Hit NEXT.

  27. Select "Replace existing Driver"

  28. NEXT

  29. Now type in a name for the printer. You could use "HP Laserjet 4100 PCL 6" if you want to.

  30. Printer sharing - Do not share the printer. As you really don't need to. Everyone can get to the printer using the method contained in the summary just as you are doing now..

  31. NEXT

  32. Print a TEST page - NO, do that later.

  33. Whew - you are finished.. Select FINISH

  34. If you made this printer your default printer, then bring up Internet Explorer and try to print a test page.. Do a FILE then PRINT...   Do you see the printer you just created - Click the mouse on it and click PRINT.

  35. You should see a little printer on the lower right portion of your taskbar.  If all is well, it will only be there for a short time and be gone.

  36. Please go down to get your first output from the printer. It is located near Peggy round the corner and near the Xerox copier.

HP 4100N Printer Configuration

Windows 98 or ME on a laptop or similar only

 

We have been relying on the Jetdirect EIO card in the HP 4100N printer to supply us with a nice embedded print spooler.  This works fine for Windows2000 which supports the concept of an HP printer port.

Windows 98 and Windows ME do not have these features.

To use the printer from these systems, you can share the printer from your windows 2000 Dell computer (or similar) and use the shared printer to gain access. There are certain things you have to set up first.

  1. Your laptop must be configured to use DHCP and it must get it's IP address automatically from our DHCP server.

  2. You must have the Windows 98 or Windows ME driver for the HP 4100N on your laptop and ready to be used for installation.

  3. IMPORTANT!! This is the bugaboo. You typed in a username a long time ago when you first setup windows 98 or ME  on your laptop.  That username MUST BE the exact same one you use in your Windows 2000 computer.  This then, establishes a trust relationship between the laptop and your Windows 2000 machine. Due to Win2000 using encrypted passwords (and the inability of windows ME) to use this feature, the way around this is keeping the usernames matching.

  4. You now share the printer on your Windows2000 machine. Give it a nice short descriptive name you can use for step 7.

  5. On your Windows 98 or ME machine, you go to the printers folder and select NEW PRINTER.

  6. You choose NETWORK printer.

  7. for the name of the network printer you type in this:  \\nameofyourcomputer\nameofprinter Where name of your computer is the name which shows up in the BigSur workgroup. Name of the printer is the name the printer was given when you shared it in step 4.

  8. For the driver, you now select HAVE DISK and point the install program to the area of the driver where you put it when you downloaded and expanded it. Probably C:\hp844en, where 844 is some model number internal to Hewlett Packard. If you need the driver for Windows ME download it from the BigSur Web server by clicking HERE.  Once you download the driver, it is in zipped .exe form. Double click on it and it will expand to the directory as in step 8.

  9. Now, keep answering questions and hitting NEXT until you get to Finish.

  10. Make sure your main Windows 2000 computer is ON when you are trying to print from your other computer which we just setup.

  11. Try a test print from Internet Explorer.

Personal Web Server

All computers with Windows 2000 Professional have personal web servers built in. It may not be installed. To install it:

  1. Control Panel

  2. Add/Remove programs

  3. Add/Remove Windows Components

  4. Check the checkbox for "Internet Information Services" IIS

  5. Hit Next and let it install

  6. Finish.

  7. Probably a good idea to reboot.

  8. This process creates a Inetpub folder on your C:\ drive.

  9. In the Inetpub folder is a wwwroot folder.

  10. Put an index.html file into the location C:\Inetpub\wwwroot folder.

  11. The index.html is the first webpage your website will display

  12. Use Microsoft Front Page or Netscape Composer to modify the index.html page.

  13. Tell your IIS webserver to use the index.html page. To do this firstly go to control panel

  14. From control panel open Administrative Tools folder

  15.  Double click on the "Personal Web Manager"

  16. Click on "ADVANCED" on the left

  17. For default documents, erase all of them and enter "index.html" without the "".

  18. Close the perweb manager.

  19. Test your Webserver!  Enter the name of your computer as it appears in the bigsur Network. For example for jefffall you would enter jefffall at the Address area of Internet Explorer.

  20. Add content to your webpage and help BigSur grow by allowing all of us to educate and help eachother via information exchange.
