Design basic State Transition diagrams with

SD Designer

I found this on the net while looking for State Diagram tools.  It will actually

produce HDL, but I have never tested the code it produces.

 

This tool is useful for drawing basic state diagrams of SCSI, iSCSI, ect.

It may have other uses.  If you find out it can do more, please publish it on your webpage.

I found the installation ZIP file has a problem.  You have to rename one of the files after the package is unzipped to make it work correctly.

So, I am having you download three files into a directory. Then run setup.

Click on the links below to download the files.. Get files one, two and three.

To download, click on each one of these sequentially:

SD Designer file #1 (sddesigner.cab)

SD Designer file #2 (setup)

SD Designer file #3 (SETUP.LST)

IMPORTANT - Setup.lst will come up in your word processor. Save it to disk

as SETUP.LST where you put the other files. Make sure it is called setup.lst and not setup.lst.txt or setup.lst.doc. Hint - use a COMMAND Window.. Use the rename command like this: rename setup.lst.txt setup.lst

Now, run setup.
